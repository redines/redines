# Fetching Guide

Use this page to fetch profile content from this repository in either rendered or raw form.

## Canonical Paths

- Profile hub: `docs/index.md`
- Professional summary: `docs/professional-summary.md`
- Current focus: `docs/current-focus.md`
- Projects: `docs/projects.md`

## GitHub Rendered URLs

Profile-repo examples for this setup (`redines/redines`):

- `https://github.com/redines/redines/blob/main/docs/index.md`
- `https://github.com/redines/redines/blob/main/docs/professional-summary.md`
- `https://github.com/redines/redines/blob/main/docs/current-focus.md`
- `https://github.com/redines/redines/blob/main/docs/projects.md`

Generic pattern:

- `https://github.com/<user>/<repo>/blob/main/docs/<page>.md`

## Raw Markdown URLs

- `https://raw.githubusercontent.com/redines/redines/main/docs/index.md`
- `https://raw.githubusercontent.com/redines/redines/main/docs/professional-summary.md`
- `https://raw.githubusercontent.com/redines/redines/main/docs/current-focus.md`
- `https://raw.githubusercontent.com/redines/redines/main/docs/projects.md`

Generic pattern:

- `https://raw.githubusercontent.com/<user>/<repo>/main/docs/<page>.md`

## Notes For Consumers

- Raw endpoints may be cached briefly, so updates can appear with a short delay.
- Keep filenames stable if other systems depend on these URLs.
- Prefer linking to `docs/index.md` as the canonical entrypoint.
